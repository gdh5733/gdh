# Mysql数据库

## 1.SQL 在机器上的读取顺序

```mysql
1.FROM  <left_table>

2.ON <join_condition>

3.<join_type> JOIN <right_table>

4.WHERE <where_condition>

5.GROUP BY <group_by_list>

6.HAVING <having_condition>

7.SELETE

8.DISTINCT <select_list>

9.ORDER BY <order_by_condition>

10.LIMIT <limit_number>
```
